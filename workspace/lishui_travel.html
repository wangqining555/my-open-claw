4月4日（周六）— 缙云仙都 + 河阳古村

清明节 · 正式旅行第一天
🌿 山水诗意
08:00
🚗
杭州出发，自驾前往缙云
从余杭区上高速：杭长高速 → 沪昆高速 → 长深高速，全程约 180km，预计 2.5 小时。清明高速免费，注意避开早高峰尽早出发。
高速费约 ¥120 预计抵达 10:30
10:30
🏞️
鼎湖峰（仙都景区）⭐ 必打卡
《道士下山》取景地，拔地而起的孤峰直插云天，峰顶有老子庙，传说黄帝炼丹升天之地。乘坐索道上山可俯瞰整个仙都景区，春日山花烂漫。
门票 ¥60 + 索道 ¥40 游玩约 2.5 小时 核心景点
🅿️ 停车
景区门口大型停车场，停车费 ¥10/次
13:00
🍜
缙云县城午餐 — 缙云烧饼 + 敲肉羹
缙云是"中国小吃文化名城"，必吃：缙云烧饼（梅干菜肉饼，外脆里嫩）、敲肉羹（土猪肉手工敲打）、土爽面、豆腐丸。推荐到人民医院附近老店或广场路老街寻味。
人均 ¥30-50 缙云特色
14:30
🏘️
河阳古村（AAAA景区）🏆
千年古村，始建于五代末期，有"浙江第一古村"之称。完整保存明清古建筑群，1600余间房屋，10余座古祠堂。推荐游览：答樵路（"十八间"古街）、循得桥、虚竹公祠。石雕、砖雕、木雕工艺精湛。
门票 ¥40 游玩约 1.5-2 小时 古村爱好者必去
17:00
🏨
返回艾克酒店（缙云扬帆商业广场店）
从河阳古村自驾约15分钟到达酒店。办理入住后可在房间休息，或逛逛商业广场周边。晚餐可在广场内或附近农家乐。
酒店就在扬帆商业广场
18:30
🌙
晚餐 + 缙云夜游
推荐尝试缙云本地菜：清炖溪鱼、霉干菜扣肉、盐卤豆腐。餐后可沿好溪边散步，夜晚两岸灯光步道颇为惬意。
人均 ¥60-80

4月5日（周日）— 缙云 → 云和梯田

旅行第二天 · 退房后前往云和
🌾 梯田云海
08:00
☀️
酒店早餐 + 退房
艾克酒店退房，行李放车上，轻装出发。早餐后直接前往云和梯田，约 120km，2小时车程。
长深高速，约 ¥60高速费
10:00
🌾
云和梯田（AAAAA景区申报中，核心为AAAA）⭐⭐ 必打卡
华东最大梯田群，被誉为"中国最美梯田"之一。清明时节灌水期，梯田如千万面镜子倒映天空与山峦，晨雾翻涌，宛如天上人间。此行最壮观景点！
门票 ¥80（网上提前购 ¥70） 强烈推荐 建议游玩 3-4 小时
🅿️ 停车
景区入口停车场 ¥10/次，可开车至山顶观景台
⏰ 最佳游玩时间提示
上午 9:00-11:00 光线最佳，看到云海概率最高！清明时节云雾较多，很大概率遇到云海奇观。
10:30
📸
云和梯田核心观景路线
推荐游览顺序：九曲云环（主入口/观景台）→ 观景台俯瞰灌水梯田 → 七星墩（海拔最高观景点，视野最开阔）→ 坑根石寨（800年古村）→ 白银谷（竹海峡谷）。可自驾至各观景点，或乘景区接驳车。
九曲云环 · 核心观景 七星墩 · 海拔最高 坑根石寨 · 古村 白银谷 · 峡谷竹海
13:00
🍜
景区内午餐（坑根石寨附近农家乐）
梯田景区内农家乐（坑根石寨附近），推荐：笋干烧肉、田螺塞肉、溪鱼干、糯米糕、蕨菜。食材取自山野，新鲜地道。
人均 ¥50-70 山野农家菜
14:30
🏡
坑根石寨（古村）— 800年历史
藏在梯田深处的古老村落，有800多年历史。石板路、老土屋、清泉潺潺，非常原生态。在这里喝一杯云和老茶，歇歇脚，感受慢下来的时光。
包含在景区门票内 悠闲漫步 1小时
16:30
🚗
前往观云居民宿入住 — 崇头镇下垟村66号
从梯田景区九曲云环驾车约20分钟（15km）直达下垟村村口。民宿就在梯田边上，窗外就是连绵梯田和竹林，视野绝佳。入住后在民宿周边散步，看夕阳下的金色梯田。
约 15km，25分钟 民宿就在梯田核心区
18:30
🌄
民宿晚餐 + 观星赏月 🌟
观云居民宿大概率提供地道农家菜，山野风味。夜晚山区光污染少，晴好天气满天繁星🌟，清明假期月色也不错。早点休息，为次日早起看云海日出养精蓄锐！
人均 ¥60-80 山里食材新鲜

早起看云海日出 ⭐⭐⭐ 全程最高光时刻
民宿就在梯田边！步行5分钟即可到达九曲云环观景台。清晨5:30起床，步行至观景台，找个好位置等待日出。晨曦中金色阳光洒在灌满水的梯田上，云雾在山谷间翻涌——这将是你们此行最难忘的画面！云和梯田春季云海出现概率约60%，清明时节极高！
强烈建议！全程最值景点！ 05:30起床，05:50前到观景台

白银谷（竹海峡谷）
梯田景区内秀丽峡谷，因古代银矿遗址而得名。峡谷两侧竹海茫茫，溪水潺潺，全程步道约2公里，轻松休闲，负氧离子爆表。早上光线穿过竹海，光影斑驳极美，且人少清净。
包含在景区门票内 游玩约 1.5 小时 轻松悠闲
💡 建议游览顺序
退房行李放车上 → 先去白银谷（人少光美）→ 回民宿取行李出发 → 县城午餐
11:30

云和县城午餐 + 采购特产

📌 出行注意事项

🎒 行李建议

  • 外套（山区昼夜温差大，清晨看日出较凉）
  • 防滑运动鞋（山路、梯田田埂）
  • 防晒霜、墨镜（白天日照较强）
  • 雨伞或雨衣（清明雨水较多）
  • 充电宝（山区充电不便）
  • 常用药品（肠胃药、创可贴）
  • 身份证、驾照、车险单

🚗 自驾提示

  • 清明高速免费，但返程（4/6）下午会有拥堵，建议13:00-14:00前出发
  • 山区弯道多，注意控制车速，不要借道超车
  • 导航提前下载离线地图，山区信号不稳定
  • 云和梯田景区内可开车至山顶，但山路较窄，小心驾驶
  • 观云居民宿提前联系房东确认停车位置

📸 拍照建议

  • 云海日出（D3早晨5:30）是最值得拍的画面，提前占好观景台位置
  • 灌水期梯田（4月）上午9-11点光线最佳，倒影最清晰
  • 河阳古村建议下午拍摄，光线柔和，古建筑更有质感
  • 白银谷上午光线穿过竹海，光影斑驳极美

🏨 住宿提示

  • 艾克酒店（缙云）4月5日退房后行车李继续前往云和
  • 观云居民宿（D2晚）海拔较高，早晚凉爽，日出需早起
  • 民宿为农家乐风格，设施简单但氛围好，建议自带洗漱用品
  • 民宿晚餐提前和老板说好，方便准备新鲜食材
