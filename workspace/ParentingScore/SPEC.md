# 育儿方式打分 App - 规格说明书

## 1. 项目概述
- **项目名称**: 育儿打分 (Parenting Score)
- **类型**: Android 原生应用
- **核心功能**: 输入孩子行为，评估家长教育方式，给出分数和建议

## 2. UI/UX 设计

### 页面结构
- **单页面应用** - 所有功能在一个界面完成
- **简洁现代设计** - Material Design 3

## 3. 功能规格

### 核心功能
1. **行为输入** - 用户输入孩子的不良行为或良好行为
2. **方式选择** - 选择家长采取的教育方式
3. **智能评分** - 根据行为类型+教育方式给出0-100分
4. **建议生成** - 根据分数给出改进建议

### 评分逻辑
- 良好行为 + 表扬 = 90-100分
- 良好行为 + 讲道理 = 80-90分
- 不良行为 + 惩罚 = 60-80分
- 不良行为 + 讲道理 = 70-85分
- 不良行为 + 忽视 = 20-40分
- 不良行为 + 表扬 = 10-30分

### 建议内容
- 90+: 优秀做法，继续保持
- 70-89: 良好，需要改进
- 50-69: 一般，需要学习
- 0-49: 需要调整教育方式
